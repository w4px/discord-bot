🤖 # بوت ديسكورد احترافي عربي

بوت ديسكورد متطور وشامل يتضمن جميع الميزات الضرورية!

## ✨ الميزات الرئيسية

### 1️⃣ **نظام الإدارة**
- `بان` - حظر عضو من السيرفر
- `كيك` - طرد عضو من السيرفر
- `تايمأوت` - إسكات عضو مؤقتاً
- `إزالة_تايمأوت` - إزالة الإسكات
- `رول` - إعطاء أو إزالة رتبة

### 2️⃣ **نظام الاقتصاد والأموال**
- `رصيد` - عرض الأموال والذهب
- `تحويل` - تحويل أموال لعضو آخر
- `يومي` - المكافأة اليومية

### 3️⃣ **الألعاب**
- `زهر` - لعبة النرد
- `قلب` - قلب العملة
- `حجركاغذمقص` - حجر كاغذ مقص
- `تعدين` - لعبة التعدين
- `ماكينة` - ماكينة القمار

### 4️⃣ **المميزات الإضافية**
- نظام الستريك اليومي
- نظام الردود التلقائية
- ترحيب الأعضاء الجدد
- نظام اللوقات والعدّاد

## 📋 المتطلبات

- Node.js 16.9.0 أو أحدث
- MongoDB (قاعدة بيانات)
- Discord Bot Token

---

## 🚀 التثبيت

### 1. استنساخ المستودع
```bash
git clone https://github.com/w4px/discord-bot.git
cd discord-bot
```

### 2. تثبيت المكتبات
```bash
npm install
```

### 3. إعداد ملف .env
انسخ `.env.example` إلى `.env` وأضف بيانات الاتصال:

```env
DISCORD_TOKEN=your_token_here
MONGODB_URI=mongodb+srv://username:password@cluster.mongodb.net/discord-bot
PREFIX=.
```

### 4. تشغيل البوت
```bash
npm start
```

---

## 🌐 نشر على Render

### 1. أنشئ حساب على [Render.com](https://render.com)

### 2. اربط مستودعك على GitHub

### 3. أنشئ خدمة جديدة:
- Build Command: `npm install`
- Start Command: `npm start`

### 4. أضف المتغيرات البيئية:
- `DISCORD_TOKEN`
- `MONGODB_URI`
- `NODE_ENV = production`

### 5. شغل الخدمة! 🚀

## 🛠️ الأوامر

### أوامر الإدارة:
```
/بان @عضو [السبب]
/كيك @عضو [السبب]
/تايمأوت @عضو المدة [السبب]
/إزالة_تايمأوت @عضو
/رول [إعطاء/إزالة] @عضو @الرتبة
```

### أوامر الاقتصاد:
```
/رصيد [@عضو]
/تحويل @العضو المبلغ
/يومي
```

### الألعاب:
```
/زهر الرقم المبلغ
/قلب الاختيار المبلغ
/حجركاغذمقص الاختيار المبلغ
/تعدين
/ماكينة المبلغ
```
